# Design

## Theme colors

From [open-props](https://open-props.style/#colors).

Used in logo:

```css
var(--violet-5, #845ef7)
var(--blue-5, #339af0)
var(--indigo-5, #5c7cfa)
```

## Logo

Created in svg with font Noto Sans and converted to path with inkscape.

```sh
inkscape design/logo.svg --export-text-to-path --export-filename _assets/logo.svg
```
