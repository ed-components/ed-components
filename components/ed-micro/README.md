## Usage


```html
<ed-micro label="Awesome micro-content">
## My micro content

It has been proven that is better to chunk content to faciliate memorization.

## Split content

Ok.
</ed-micro>
```

If no label is provided it will use first child as label.

```html
<ed-micro>
## My micro content

It has been proven that is better to chunk content to faciliate memorization.

## Split content

Ok.
</ed-micro>
```
